# Cloud Synced Recorder

## Setup

The Cloudinary credentials are pre-configured in the code for this demo. You can replace them with your own credentials if needed.

## Cloud Integration

- Uses Cloudinary for storage.
- Uploads are handled by WorkManager (UploadWorker).
- Requires internet permission.
- Credentials are securely stored in CloudinaryHelper.
