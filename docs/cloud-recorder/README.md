# Cloud-Synced Field Recorder

A voice/meeting recorder that uploads audio to Cloudinary.

## Features
- Audio recording (MediaRecorder)
- Local storage
- Background upload (WorkManager - coming soon)

## Architecture
Clean Architecture

## Tech Stack
- MediaRecorder
- ViewModel
- WorkManager
- Cloudinary

## How to Test
1. Open app
2. Grant permissions
3. Click Record

## Note
Upload feature requires Cloudinary credentials.
